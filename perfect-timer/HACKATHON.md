# Hackathon Submission Guide

## Project: Perfect? ⏱️

A precision timing game where players stop a timer at exact moments to progress through levels. Built on Base with WalletConnect integration.

## Hackathon Compliance

### Base Hackathon Requirements ✅

**Leaderboard Tracking:**
- ✅ **Fees generated by deployed contracts**: Smart contract deployed on Base generates gas fees with each score submission
- ✅ **Mini App Developer Rewards**: Built as a Base Mini App using `create-onchain --mini`
- ✅ **GitHub contributions**: Public repository with active development

**Implementation:**
- Contract: `PerfectLeaderboard.sol` deployed on Base
- Mini App: Configured via `minikit.config.ts`
- OnchainKit: Integrated for wallet and Base features

### WalletConnect Hackathon Requirements ✅

**Leaderboard Tracking:**
- ✅ **Integration of WalletKit SDK/Reown AppKit**: Using `@reown/appkit@^1.8.14` and `@reown/appkit-adapter-wagmi@^1.8.14`
- ✅ **GitHub contributions**: Public repository
- ✅ **Fees generated by smart contracts**: Same contract as Base submission

**Implementation:**
- Reown AppKit: Configured in `app/config/wagmi.ts`
- Multi-wallet support via AppKit
- Wagmi integration for contract interactions

## Key Features

1. **Timer Game Mechanics**
   - Progressive difficulty (target time decreases each level)
   - Precision scoring (within 100ms = success)
   - Real-time feedback with accuracy percentage

2. **On-Chain Integration**
   - Leaderboard stored on Base blockchain
   - Score submission via smart contract
   - Wallet-based player identification

3. **Web3 Features**
   - Multiple wallet support (via Reown AppKit)
   - Base network integration (via OnchainKit)
   - Transaction handling for score submission

## Tech Stack

- **Frontend**: Next.js 15, React 19, TypeScript
- **Web3 Libraries**: 
  - OnchainKit (Base integration)
  - Reown AppKit (WalletConnect)
  - Wagmi + Viem (Ethereum interactions)
- **Smart Contracts**: Solidity 0.8.24, Hardhat
- **Styling**: CSS Modules

## Deployment Instructions

### 1. Environment Setup
```bash
# Install dependencies
npm install

# Configure .env
NEXT_PUBLIC_ONCHAINKIT_API_KEY="..."
NEXT_PUBLIC_REOWN_PROJECT_ID="..."
PRIVATE_KEY="..."
```

### 2. Deploy Smart Contract
```bash
# Testnet
npm run deploy:sepolia

# Mainnet
npm run deploy:base
```

### 3. Deploy Application
```bash
# Build
npm run build

# Deploy to Vercel
vercel deploy --prod
```

## Demo Flow

1. User visits the app
2. Connects wallet via Reown AppKit (multiple wallet options)
3. Plays the timing game
4. Submits high score to on-chain leaderboard
5. Views global leaderboard
